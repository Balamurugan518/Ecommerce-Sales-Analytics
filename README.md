# Ecommerce Sales Analytics

A comprehensive SQL-based analytics project designed to analyze ecommerce business data using MySQL. This project demonstrates database design, data management, and business intelligence reporting through advanced SQL queries and views.

## Project Objectives

- Design and implement a relational ecommerce database.
- Store and manage customer, product, order, and payment information.
- Perform business analysis using SQL queries.
- Generate insights on customer behavior, sales performance, and revenue trends.
- Create reusable database views for reporting purposes.

## Database Schema

The project consists of the following tables:

| Table | Description |
|---------|-------------|
| customers | Stores customer information |
| products | Stores product details |
| orders | Stores order records |
| order_items | Stores products associated with each order |
| payments | Stores payment transaction details |
